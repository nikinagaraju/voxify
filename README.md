# 🎬 Voxify

Voxify is an AI-powered YouTube video transcription and Q&A app built with Streamlit.

## Features
- 🎵 Download audio from any YouTube video
- 📝 Transcribe audio to text using Whisper AI
- 🌐 Translate transcript to Hindi or Urdu
- 🔊 Generate audio from transcript using gTTS
- 🤖 Ask questions about the video using Groq AI
- 🔍 Semantic search using FAISS

## Tech Stack
- Streamlit
- Faster Whisper
- Sentence Transformers
- FAISS
- Groq API
- gTTS
- yt-dlp

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key in `.env` file
4. Run: `streamlit run app.py`
